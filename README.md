# PokeShop
Proyecto integrador final Landing page

Link Vercel: https://poke-shop-teal.vercel.app/

## Run
```
    npm install
    npm run start
```

```
    yarn install
    yarn run start
```
Starts on `http://localhost:3000`

## Build
```
    npm run build
```

## Development technologies and external packages
* React 
* Jest & React testing library
* React Router
* Styled Components
* Node sass
* Axios
* Material UI
* Redux
